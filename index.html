How to use the tool?
  • Click on the new design file button to create a new design file.
  • Click on the open design file button to open an existing design file.
  • Click on the export design button to export the design file.
  • Click on the help button to open the help menu.
  • Click on the layer to select object from canvas.
  • Click on any object in canvas to open the design menu.
  • Click on the square button to create a square.
  • Click on the rectangle button to create a rectangle.
  • Click on the polygon button to create a polygon.
  • Click on the three dots button (if exists) in design menu and delete button to delete the selected item.
  • Click on the create only convex hull vertices checkbox to create only convex hull vertices.
  • To draw a polygon, click on the canvas to create a vertex and then click on the canvas again to create another vertex. To finish the polygon, right click on the canvas.
